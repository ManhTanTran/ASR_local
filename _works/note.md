# Notes

Scratchpad cho các ghi chú nhanh của project ASR.

## Hiện trạng

- `ASR/` là repo main clone từ GitHub.
- `ASR_local/` giữ phần làm việc cá nhân: notebook cuối, src tách riêng, và work notes.
- Notebook cuối nằm ở `notebooks/final/`.
- Logic notebook đã tách sang `src/asr_local/`.

## Việc cần nhớ

- Không để credential Kaggle/token trong repo.
- Khi chạy Kaggle thật, kiểm tra biến `RUN_REMOTE` trong notebook FastConformer.
- Các work note trong folder này là ghi chú cá nhân, không phải docs chính thức của repo.
