# _works

Khu vực ghi chú công việc cá nhân cho project ASR.

Folder này thay cho `_docs` cũ. Nội dung được dựng lại từ cấu trúc folder đã thấy trong log làm việc, không phải bản phục hồi nguyên văn từ file gốc.

## Cấu trúc

- `note.md`: scratch note chung.
- `00_daily/`: nhật ký làm việc theo tháng.
- `00_general/`: ghi chú setup, git, module Python.
- `02_fastconformer_finetune/`: ý tưởng và issue cho nhánh fine-tune FastConformer/VIVOS.
