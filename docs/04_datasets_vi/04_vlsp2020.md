# 04 — VLSP 2020 (VinAI 100h)

Bộ 100h do **VinAI / VinBigData** chia sẻ cho cộng đồng trong khuôn khổ workshop **VLSP 2020**
(Vietnamese Language and Speech Processing).

---

## Số liệu

- **Loại:** đọc văn bản — chủ yếu **tin tức, thông báo chính thức** (giọng trang trọng).
- **Giờ audio:** ~**100h**.
- **Số mẫu:** ~**56.427**.
- **Sample rate / định dạng:** trên HF lưu **parquet** (auto-convert). Sample rate gốc **cần kiểm chứng** (đo file thật).
- **Transcript:** có (độ chính xác transcript ~96% theo người làm mirror — có chỗ chưa chuẩn hoá tên riêng/từ nước ngoài).
- **License:** **CC BY 4.0** — **cho phép thương mại** (theo thẻ HF mirror; nếu dùng cho sản phẩm nên đối chiếu lại điều khoản VLSP gốc cho chắc).

## Convert sang manifest NeMo

Bản parquet trên HF có cột `audio` (array + sampling_rate) + `transcription`. Pattern giống VIVOS:

```python
import json, os, soundfile as sf
from datasets import load_dataset

OUT = "data/manifests/vlsp2020/train.jsonl"
RAW = "data/raw/vlsp2020/wav"
os.makedirs(os.path.dirname(OUT), exist_ok=True); os.makedirs(RAW, exist_ok=True)

ds = load_dataset("doof-ferb/vlsp2020_vinai_100h", split="train")
with open(OUT, "w", encoding="utf-8") as w:
    for i, row in enumerate(ds):
        arr, sr = row["audio"]["array"], row["audio"]["sampling_rate"]
        text = (row.get("transcription") or "").strip()
        if not text:
            continue
        path = os.path.join(RAW, f"vlsp_{i:06d}.wav")
        sf.write(path, arr, sr)
        w.write(json.dumps({
            "audio_filepath": path,
            "duration": round(len(arr) / sr, 3),
            "text": text,
        }, ensure_ascii=False) + "\n")
```

## Lưu ý

- Giọng **tin tức trang trọng** → khác giọng nói chuyện tự nhiên của khách gọi callbot. Dùng để fine-tune nền tốt,
  nhưng đừng kỳ vọng nó "dạy" được văn nói đời thường.
- Transcript ~96% đúng → có nhiễu nhãn nhẹ, ảnh hưởng nhỏ khi đo WER.
